## 📝 Project Overview
**व्यापारी मित्र** एक Spring Boot आधारित REST API है जो छोटे व्यापारियों को उनके व्यवसाय को डिजिटल रूप से प्रबंधित करने में मदद करता है। यह प्रोजेक्ट विशेष रूप से मराठी भाषा को सपोर्ट करता है।

## ✨ Key Features

### 1. **Authentication & Authorization** 🔐
- JWT based authentication
- User registration and login
- Role-based access control

### 2. **Customer Management** 👥
- Add/Update/Delete customers
- Customer details with contact information
- Customer transaction history

### 3. **Transaction Management** 💰
- Record daily transactions
- Income/Expense tracking
- Transaction history and filtering

### 4. **Dashboard & Reports** 📊
- Real-time business dashboard
- Sales and revenue reports
- Daily/Monthly/Yearly reports
- Custom report generation

### 5. **File Upload & Storage** 📁
- Upload invoices and documents
- File storage management
- Support for multiple file types

### 6. **Reminder System** ⏰
- Payment reminders
- Due date notifications
- Custom reminders for customers

### 7. **Search Functionality** 🔍
- Search customers by name/phone
- Search transactions by date
- Advanced filtering options

### 8. **Settings Management** ⚙️
- Business profile settings
- Tax and GST configuration
- Notification preferences

### 9. **Multi-language Support** 🌐
- Marathi language support (`MarathiMessages`)
- English language support
- Easy language switching

## 🛠️ Technology Stack

| Layer | Technology |
|-------|------------|
| **Framework** | Spring Boot |
| **Security** | Spring Security + JWT |
| **Database** | MySQL, JPA/Hibernate |
| **File Storage** | Local file system |
| **API Docs** | Swagger/OpenAPI |
| **Reporting** | Custom Report Generator |
| **Build Tool** | Maven |
| **Java Version** | Java 17 |
